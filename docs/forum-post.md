## ✈️🚗 MikeScript Gamepad — fly and drive in FiveM with your flight stick

Ever wanted to fly a plane or heli in FiveM with an actual **joystick / HOTAS**
instead of a keyboard? FiveM (like GTA V) reads **Xbox controllers** but ignores
flight sticks. **MikeScript Gamepad** sits in the middle: it reads your stick and
feeds the game a **virtual Xbox controller**, with a simple app to map your
controls however you like.

It's **free**, open, and works with basically any controller.

### ✨ What it does
- Turns **any joystick / HOTAS / throttle** into an Xbox controller the game understands
- Live mapping screen — **see every axis & button react** so setup is obvious
- One-click **Auto-map** for any stick, then fine-tune
- Proper **throttle, brake & reverse** handling (push to go, button to brake/back up)
- **Rudder/yaw**, deadzone, invert, sensitivity — all adjustable
- Save & share your setups as simple profile files
- Even supports **multiple controllers at once** (separate stick + throttle + pedals, mixed brands)

### 🎮 Works with your gear
Auto-recognizes **Thrustmaster, Logitech / Saitek, Turtle Beach, VKB, VIRPIL,
Honeycomb, WINWING, MOZA, CH Products, 8BitDo** — and anything else falls back to
a generic layout that still works. Ready-made profiles included for several popular sticks.

### 🚀 Setup (about 2 minutes)
1. Download and unzip.
2. Double-click **`Install.bat`** once (installs Python + the ViGEmBus driver + packages — click **Yes** on any prompts).
3. Plug in your stick, double-click **`MikeScript-Gamepad.bat`**.
4. Pick your device → **Auto-map** → sweep the throttle once → **Start Bridge**. Go fly. ✈️

### ❓ Is it safe / will I get banned?
It's a standalone **input remapper** — the same idea as reWASD, JoyToKey, or
Steam Input. It creates a virtual Xbox controller from **your** real inputs using
the well-known **ViGEmBus** driver. It does **not** touch the game, inject, read
memory, or automate anything (no aimbot/macros). That said, every server sets its
own rules — if you're unsure, ask your server's staff.

### 🛠️ Notes
- Windows 10/11. The download is unsigned, so SmartScreen may warn — that's normal for any small free tool.
- Some sticks (e.g. Thrustmaster Hotas One) also show up as an Xbox pad; if the game sees double inputs, hide the physical one with **HidHide**.
